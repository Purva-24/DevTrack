# DevTrack
> A simple and clean daily task tracker built with pure HTML & CSS — choose your tasks, track your progress, and stay on top of your day!

---

## 📌 Project Overview

**Daily Task Tracker** is a static web app built using only HTML and CSS. It allows users to select their daily tasks and visually track which ones have been completed. The project is focused on simplicity and clean UI design — helping users stay organized without any complexity.

---

## ✨ Features

- 📝 **Task Selection** — Choose from a list of daily tasks you want to complete
- ✅ **Task Completion Tracking** — Mark tasks as done and visually see your progress
- 📊 **Progress Overview** — See how many tasks are completed vs remaining
- 🎨 **Clean UI** — Minimal and distraction-free interface
- 📱 **Responsive Design** — Works smoothly on desktop and mobile screens

---

## 🛠️ Built With

- **HTML5** — Page structure and task layout
- **CSS3** — Styling, progress indicators, and responsiveness

> No JavaScript. No frameworks. No backend. Pure HTML & CSS only.

## 🚀 How to Run

No installation or setup needed!

1. Download or clone the repository
2. Open `index.html` in any web browser
3. That's it — start tracking your day!

```bash
git clone https://github.com/your-username/daily-task-tracker.git
cd daily-task-tracker
# Just open index.html in your browser
```
